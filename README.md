# movie-info-registry
本项目用于构建一个影视类网站信息登记示例，通过文字方式记录不同影视站点的基础情况与访问状态。仓库内容注重信息的完整性与可读性，方便长期维护与更新。所有记录均来源于公开页面，仅用于信息整理与参考，不构成任何形式的推荐或合作关系。
